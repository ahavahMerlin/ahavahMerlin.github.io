Html
Entendendo o código HTML:
<form action="processar_formulario.php" method="post" enctype="multipart/form-data">: Esta é a tag principal do formulário.
action="processar_formulario.php": Especifica o arquivo PHP que processará os dados do formulário.
method="post": Define que os dados serão enviados via método POST (mais seguro para informações sensíveis).
enctype="multipart/form-data": Essencial para permitir o envio de arquivos (imagens, etc.).
<label>: Cria rótulos para cada campo, indicando o que o usuário deve preencher.
<input type="text">: Cria campos de texto para a maioria das informações.

<input type="tel">: Cria campos para números de telefone (com teclado numérico em dispositivos móveis).
<input type="file">: Cria campos para o usuário selecionar arquivos (imagens).
accept="image/*": Restringe a seleção a arquivos de imagem.
<textarea>: Cria um campo de texto de múltiplas linhas (para "Horários de Funcionamento").
<button type="submit">: Cria o botão de envio do formulário.
Passo 2: Criando o Arquivo PHP para Processar o Formulário
Crie um arquivo com a extensão .php (ex: processar_formulario.php).
Cole o código PHP abaixo. Importante: Adapte as configurações de e-mail ($seu_email, $smtp_server, $smtp_user, $smtp_senha) com os dados do seu servidor de e-mail.
<?php

use PHPMailer\PHPMailer\PHPMailer;
use PHPMailer\PHPMailer\Exception;

require 'vendor/autoload.php'; // Ajuste o caminho se necessário

// Configurações de e-mail (ALTERE ESTES VALORES)
$seu_email = 'seu_email@exemplo.com'; // Seu endereço de e-mail
$smtp_server = 'smtp.exemplo.com'; // Servidor SMTP
$smtp_user = 'seu_usuario_smtp'; // Usuário SMTP
$smtp_senha = 'sua_senha_smtp'; // Senha SMTP
$smtp_port = 587; // Porta SMTP (geralmente 587 ou 465)
$email_destino = 'email_para_receber@exemplo.com'; // E-mail para onde enviar os dados

// Função para tratar erros de upload
function tratarErroUpload($error) {
    switch ($error) {
        case UPLOAD_ERR_OK:
            return null; // Sem erros
        case UPLOAD_ERR_INI_SIZE:
            return 'O arquivo excedeu o tamanho máximo permitido (configurado no php.ini).';
        case UPLOAD_ERR_FORM_SIZE:
            return 'O arquivo excedeu o tamanho máximo permitido (configurado no formulário).';
        case UPLOAD_ERR_PARTIAL:
            return 'O arquivo foi parcialmente carregado.';
        case UPLOAD_ERR_NO_FILE:
            return 'Nenhum arquivo foi enviado.';
        case UPLOAD_ERR_NO_TMP_DIR:
            return 'Pasta temporária ausente.';
        case UPLOAD_ERR_CANT_WRITE:
            return 'Falha ao escrever o arquivo em disco.';
        case UPLOAD_ERR_EXTENSION:
            return 'Uma extensão do PHP impediu o upload do arquivo.';
        default:
            return 'Erro desconhecido no upload.';
    }
}

// Processamento do formulário
if ($_SERVER["REQUEST_METHOD"] == "POST") {

    // Coleta dos dados do formulário
    $nome_empresa = $_POST['nome_empresa'];
    $cpf_cnpj = $_POST['cpf_cnpj'];
    $url = $_POST['url'];
    $endereco = $_POST['endereco'];
    $bairro = $_POST['bairro'];
    $cep = $_POST['cep'];
    $cidade = $_POST['cidade'];
    $estado = $_POST['estado'];
    $tel_fixo = $_POST['tel_fixo'];
    $celular = $_POST['celular'];
    $tel_principal = $_POST['tel_principal'];
    $horarios = $_POST['horarios'];
    $email_pagseguro = $_POST['email_pagseguro'];
    $comissao = $_POST['comissao'];
    $nome_captador = $_POST['nome_captador'];
    $email_captador = $_POST['email_captador'];

    // Processamento dos uploads de arquivos
    $upload_dir = 'uploads/'; // Pasta onde os arquivos serão salvos
    if (!is_dir($upload_dir)) {
        mkdir($upload_dir, 0777, true); // Cria a pasta se não existir
    }

    $logo_path = '';
    if (isset($_FILES['logo']) && $_FILES['logo']['error'] == UPLOAD_ERR_OK) {
        $logo_name = uniqid() . '_' . $_FILES['logo']['name'];
        $logo_path = $upload_dir . $logo_name;
        if (!move_uploaded_file($_FILES['logo']['tmp_name'], $logo_path)) {
            $logo_path = 'Erro ao salvar o logo.';
        }
    } elseif (isset($_FILES['logo'])) {
        $erro = tratarErroUpload($_FILES['logo']['error']);
        $logo_path = 'Erro no upload do logo: ' . $erro;
    }

    $imagem_banner_path = '';
    if (isset($_FILES['imagem_banner']) && $_FILES['imagem_banner']['error'] == UPLOAD_ERR_OK) {
        $imagem_banner_name = uniqid() . '_' . $_FILES['imagem_banner']['name'];
        $imagem_banner_path = $upload_dir . $imagem_banner_name;
        if (!move_uploaded_file($_FILES['imagem_banner']['tmp_name'], $imagem_banner_path)) {
            $imagem_banner_path = 'Erro ao salvar o banner.';
        }
    } elseif (isset($_FILES['imagem_banner'])) {
        $erro = tratarErroUpload($_FILES['imagem_banner']['error']);
        $imagem_banner_path = 'Erro no upload do banner: ' . $erro;
    }

    // Montagem da mensagem de e-mail
    $mensagem = "
        <html>
        <head>
            <title>Nova Solicitação</title>
        </head>
        <body>
            <p><strong>Nome do Negócio:</strong> $nome_empresa</p>
            <p><strong>CPF/CNPJ:</strong> $cpf_cnpj</p>
            <p><strong>URL:</strong> $url</p>
            <p><strong>Endereço:</strong> $endereco</p>
            <p><strong>Bairro:</strong> $bairro</p>
            <p><strong>CEP:</strong> $cep</p>
            <p><strong>Cidade:</strong> $cidade</p>
            <p><strong>Estado:</strong> $estado</p>
            <p><strong>Telefone Fixo:</strong> $tel_fixo</p>
            <p><strong>Celular:</strong> $celular</p>
            <p><strong>Telefone Principal:</strong> $tel_principal</p>
            <p><strong>Horários:</strong> " . nl2br($horarios) . "</p>
            <p><strong>Email PagSeguro:</strong> $email_pagseguro</p>
            <p><strong>Comissão:</strong> $comissao</p>
            <p><strong>Nome Captador:</strong> $nome_captador</p>
            <p><strong>Email Captador:</strong> $email_captador</p>
    ";

    // Adiciona links para os arquivos, se foram enviados com sucesso
    if ($logo_path && strpos($logo_path, 'Erro') === false) {
        $mensagem .= "<p><strong>Logo:</strong> <a href='" . $logo_path . "'>Ver Logo</a></p>";
    } else {
        $mensagem .= "<p><strong>Logo:</strong> " . $logo_path . "</p>";
    }

    if ($imagem_banner_path && strpos($imagem_banner_path, 'Erro') === false) {
        $mensagem .= "<p><strong>Banner:</strong> <a href='" . $imagem_banner_path . "'>Ver Banner</a></p>";
    } else {
        $mensagem .= "<p><strong>Banner:</strong> " . $imagem_banner_path . "</p>";
    }

    $mensagem .= "</body></html>";

    // Envio do e-mail usando PHPMailer
    $mail = new PHPMailer(true);

    try {
        // Configurações do servidor SMTP
        $mail->SMTPDebug = 0; // 0 para desativar, 2 para debug
        $mail->isSMTP();
        $mail->Host = $smtp_server;
        $mail->SMTPAuth = true;
        $mail->Username = $smtp_user;
        $mail->Password = $smtp_senha;
        $mail->SMTPSecure = PHPMailer::ENCRYPTION_STARTTLS; // ou PHPMailer::ENCRYPTION_SMTPS
        $mail->Port = $smtp_port;

        // Configurações da mensagem
        $mail->setFrom($seu_email, 'Formulário de Solicitação');
        $mail->addAddress($email_destino); // Email do destinatário
        $mail->isHTML(true);
        $mail->Subject = 'Nova Solicitação de Anúncio';
        $mail->Body = $mensagem;
        $mail->AltBody = strip_tags($mensagem); // Texto sem HTML

        $mail->send();
        echo '<p style="color: green;">Mensagem enviada com sucesso!</p>';
    } catch (Exception $e) {
        echo "<p style='color: red;'>Erro ao enviar a mensagem: {$mail->ErrorInfo}</p>";
    }
}
?>
Use code with caution.
PHP
Entendendo o código PHP:
use ...: Importa as classes necessárias do PHPMailer.
require 'vendor/autoload.php';: Inclui o autoloader do Composer, que carrega as classes do PHPMailer. Você precisa instalar o PHPMailer via Composer (veja o próximo passo).
$seu_email, $smtp_server, $smtp_user, $smtp_senha: VOCÊ DEVE CONFIGURAR ESSAS VARIÁVEIS com os dados do seu servidor de e-mail SMTP. Se você usa Gmail, procure tutoriais específicos para "PHPMailer Gmail" (mas lembre-se que o Gmail pode ter restrições).
$_SERVER["REQUEST_METHOD"] == "POST": Verifica se o formulário foi enviado via método POST.
$_POST['nome_empresa'], $_POST['cpf_cnpj'], etc.: Coleta os dados enviados pelo formulário.
$_FILES['logo'], $_FILES['imagem_banner']: Acessa os arquivos enviados.
move_uploaded_file(): Move os arquivos para uma pasta no servidor (uploads/). Certifique-se de que essa pasta tenha permissões de escrita.
$mensagem: Monta o corpo do e-mail em formato HTML.
PHPMailer: A classe que envia o e-mail.
SMTPDebug = 2: Ativa o modo de depuração (útil para encontrar erros). Coloque 0 para desativar em produção.
isSMTP(): Indica que o envio será via SMTP.
Host, SMTPAuth, Username, Password, SMTPSecure, Port: Configurações do servidor SMTP.
setFrom(): Define o remetente do e-mail.
addAddress(): Define o destinatário do e-mail.
isHTML(true): Indica que o corpo do e-mail é HTML.
Subject: Define o assunto do e-mail.
Body: Define o corpo do e-mail (HTML).
AltBody: Define o corpo do e-mail em texto puro (para clientes de e-mail que não suportam HTML).
send(): Envia o e-mail.
Passo 3: Instalando o PHPMailer (Obrigatório)
O PHPMailer é uma biblioteca externa que facilita o envio de e-mails via PHP. Você precisa instalá-lo usando o Composer (um gerenciador de dependências para PHP):
Instale o Composer: Se você ainda não tem, baixe e instale o Composer em https://getcomposer.org/.
Abra o terminal/prompt de comando: Navegue até a pasta onde você salvou o arquivo processar_formulario.php.
Execute o comando:
composer require phpmailer/phpmailer
Use code with caution.
Bash
Isso irá baixar e instalar o PHPMailer e criar a pasta vendor (onde o PHPMailer será armazenado) e o arquivo autoload.php.
Passo 4: Ajustando as Permissões da Pasta uploads/
A pasta uploads/ precisa ter permissões de escrita para que o PHP possa salvar os arquivos enviados pelo formulário. A forma de fazer isso depende do seu sistema operacional:
Linux/macOS: Use o comando chmod 777 uploads/ no terminal. Cuidado: Em servidores de produção, permissões 777 podem ser um risco de segurança. O ideal é ajustar as permissões para que apenas o usuário do servidor web tenha permissão de escrita.
Windows: Clique com o botão direito na pasta, vá em "Propriedades", "Segurança" e adicione permissão de escrita para o usuário do servidor web (geralmente "IUSR" ou "Serviço de Rede").
Passo 5: Colocando os Arquivos no Servidor
Envie os arquivos formulario.html e processar_formulario.php para o seu servidor web.
Envie a pasta vendor (com o PHPMailer) e a pasta uploads para o mesmo diretório onde você colocou os arquivos PHP.
Acesse o arquivo formulario.html através do seu navegador (ex: www.seusite.com/formulario.html).
Testando o Formulário
Preencha o formulário com dados de teste e envie.
Verifique se você recebe o e-mail com os dados do formulário.
Verifique se os arquivos (logo e banner) foram salvos na pasta uploads/.
Se tiver algum erro, use o modo de depuração do PHPMailer ($mail->SMTPDebug = 2;) para ver as mensagens de erro e tente identificar o problema.
Dicas Adicionais:
Validação: Adicione validação no lado do cliente (JavaScript) para garantir que os campos obrigatórios sejam preenchidos e que os dados estejam em formatos corretos (ex: e-mail válido, CEP válido).
Segurança:
Use funções de sanitização para limpar os dados do formulário e evitar ataques de SQL injection e XSS.
Implemente um sistema de CAPTCHA para evitar envios automáticos (bots).
Não armazene senhas no código. Use variáveis de ambiente ou arquivos de configuração seguros.
Limite o tamanho dos arquivos que podem ser enviados.
Estilo: Use CSS para deixar o formulário mais bonito e com a identidade visual do seu site.
Mensagens de Sucesso/Erro: Melhore as mensagens de feedback para o usuário, informando se o envio foi bem-sucedido ou se ocorreu algum erro.
Criptografia: Use HTTPS no seu site para proteger os dados do formulário durante a transmissão.
Observações Importantes:
Hospedagem: Para que o PHP funcione, você precisa de uma hospedagem que suporte PHP e tenha o PHPMailer instalado (ou que permita que você instale).
Configurações do Servidor de E-mail: As configurações de SMTP (servidor, usuário, senha, porta, segurança) variam de acordo com o seu provedor de e-mail. Consulte a documentação do seu provedor ou entre em contato com o suporte técnico para obter essas informações.
Gmail: Se você usar o Gmail, pode precisar ativar a opção "Acesso a apps menos seguros" (o que não é recomendado por questões de segurança) ou usar senhas de aplicativo.
Erros: Se você encontrar erros, leia atentamente as mensagens de erro, verifique as configurações de e-mail, as permissões de arquivos/pastas e se o PHPMailer está instalado corretamente.
Lembre-se de que este é um exemplo básico. Você pode (e deve) adaptá-lo para atender às suas necessidades específicas. Se tiver alguma dúvida ou precisar de ajuda em alguma etapa, me diga!
